To read the recorded RTT you can use the MATLAB command
“importdata()”

Example
RTT=importdata('SavedRTT/filename.mat')

RTT_N100=importdata('SavedRTT/RTT_N100_22Jan110454.mat')  % filename=RTT_N100_22Jan110454
RTT_N65=importdata('SavedRTT/RTT_N65_22Jan110547.mat')     % filename=RTT_N65_22Jan110547
